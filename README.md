# Repository Structure

This repository provides the project organization for **PerGraFuse**.

## Root Files

- `configs.py`  
  Configuration file for model settings, paths, and related parameters.

- `run_train.py`  
  Entry script for model training.

- `run_fusion.py`  
  Entry script for image fusion inference/testing.

## Folders

- `fusion_result/`  
  This folder stores representative fusion results.  
  At the current stage, only a small number of sample results are provided for illustration.

- `models/`  
  This folder is used for model-related files, including network components and checkpoints.  
  The complete model implementation and trained weights are not fully released at the current stage.

- `runs/`  
  This folder is used to save training logs, intermediate outputs, and experiment records.  
  Only limited or placeholder content is currently provided.

- `utils/`  
  This folder contains utility functions and shared helper modules used in the project.

- `docs/`  
  This folder contains documentation for repository organization, code availability, and data availability.

## Current Public Release

At the current review stage, this repository only provides:
- partial project structure,
- limited illustrative code/files,
- a small number of representative fusion examples,
- documentation describing the organization of the repository.

## Future Update

The complete source code, full experimental scripts, trained model weights, and full dataset information will be released after the paper is accepted and published online.
